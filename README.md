# Analysis of Elderly Depression
노인의 우울 상태에 영향을 미치는 요인을 분석한 프로젝트를 정리한 저장소입니다.  
노인의 우울 점수와 자녀와의 교류 빈도, 사회활동 빈도 간의 관계를 분석하고, 이를 바탕으로 노인 정신건강 개선을 위한 정책적 시사점을 도출합니다.

## Project Overview

본 프로젝트는 노인의 우울 상태가 가족관계 및 사회적 활동과 어떤 관련을 가지는지 분석합니다.  
특히 자녀와의 만남, 자녀와의 연락, 사회활동 참여 빈도가 노인의 우울 점수에 미치는 영향을 살펴보았습니다.

분석의 핵심 질문은 다음과 같습니다.

- 자녀와 자주 만나는 노인은 우울 점수가 낮은가?
- 자녀와의 연락 빈도는 우울 상태와 관련이 있는가?
- 사회활동 빈도는 노인의 우울 완화에 영향을 미치는가?
- 자녀와의 교류가 사회활동의 역할을 대체할 수 있는가?
- 가족관계와 사회활동 중 노인의 우울 상태에 더 중요한 요인은 무엇인가?

## Variables

| Variable | Description |
|---|---|
| `depression` | 노인의 우울 점수 |
| `CES-D10` | 우울 상태를 측정하기 위한 척도 |
| `meet` | 자녀와의 만남 빈도 |
| `contact` | 자녀와의 연락 빈도 |
| `social` | 사회활동 참여 빈도 |

## Methodology

본 프로젝트에서는 다음과 같은 분석 절차를 수행하였습니다.

1. 데이터 전처리
   - 결측치 확인 및 처리
   - 주요 변수 선택
   - 변수 형태 점검
   - 분석 목적에 맞는 변수 재구성

2. 탐색적 데이터 분석
   - 주요 변수의 분포 확인
   - 우울 점수와 설명변수 간 관계 시각화
   - 변수 간 상관관계 확인

3. 변수 표준화
   - 자녀 만남 빈도, 연락 빈도, 사회활동 빈도의 단위 차이를 줄이기 위해 표준화 수행
   - 회귀계수 비교 가능성 확보

4. 회귀분석
   - 우울 점수를 종속변수로 설정
   - 자녀 만남 빈도, 연락 빈도, 사회활동 빈도를 설명변수로 사용
   - 각 변수가 우울 점수에 미치는 영향을 분석

5. 상호작용 효과 검토
   - 자녀 교류와 사회활동 간의 상호작용 가능성 검토
   - 자녀와의 만남이 사회활동의 역할을 대체하거나 보완하는지 분석

## Main Findings

분석 결과, 노인의 우울 상태는 단순히 자녀와의 교류만으로 설명되기 어렵고 사회활동 참여와도 밀접한 관련이 있음을 확인하였습니다.
특히 사회활동은 노인의 일상적 관계망과 외부 활동 기회를 제공한다는 점에서 우울 완화에 중요한 역할을 할 가능성이 있습니다.  
자녀와의 교류 역시 정서적 지지 측면에서 의미가 있지만, 사회활동을 완전히 대체하기보다는 함께 고려되어야 할 요인입니다.

## Policy Implications

본 프로젝트의 분석 결과를 바탕으로 다음과 같은 정책적 시사점을 제안할 수 있습니다.

- 노인의 사회활동 참여 기회를 확대해야 합니다.
- 단순한 가족 돌봄 중심 정책을 넘어 지역사회 기반의 관계망 형성이 필요합니다.
- 노인복지관, 주민센터, 지역 커뮤니티 프로그램을 활용한 정기적 사회참여 프로그램이 중요합니다.
- 자녀와의 교류가 적은 노인을 대상으로 한 사회적 고립 예방 프로그램이 필요합니다.
- 우울 고위험군 노인을 조기에 파악하고, 사회활동 참여와 연계하는 정책이 요구됩니다.
